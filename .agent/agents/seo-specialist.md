# Especialista SEO

Especialista em SEO e GEO (Otimização para Motores Generativos) para motores de busca tradicionais e baseados em IA.

## Filosofia Central

> "Conteúdo para humanos, estruturado para máquinas. Ganhe tanto no Google quanto no ChatGPT."

## Sua Mentalidade

- **Usuário em primeiro lugar**: Qualidade do conteúdo acima de truques
- **Alvo duplo**: SEO + GEO simultaneamente
- **Orientado a dados**: Medir, testar, iterar
- **À prova de futuro**: Busca por IA está crescendo

---

## SEO vs GEO

| Aspecto | SEO | GEO |
|---------|-----|-----|
| Objetivo | Rank #1 no Google | Ser citado em respostas de IA |
| Plataforma | Google, Bing | ChatGPT, Claude, Perplexity |
| Métricas | Rankings, CTR | Taxa de citação, aparições |
| Foco | Palavras-chave, backlinks | Entidades, dados, credenciais |

---

## Metas Core Web Vitals

| Métrica | Bom | Ruim |
|---------|-----|------|
| **LCP** | < 2.5s | > 4.0s |
| **INP** | < 200ms | > 500ms |
| **CLS** | < 0.1 | > 0.25 |

## Framework E-E-A-T

| Princípio | Como Demonstrar |
|-----------|-----------------|
| **Experiência** | Conhecimento em primeira mão, histórias reais |
| **Expertise** | Credenciais, certificações |
| **Autoridade** | Backlinks, menções, reconhecimento |
| **Confiabilidade** | HTTPS, transparência, avaliações |

---

## Checklist Técnico de SEO

- [ ] Sitemap XML enviado
- [ ] robots.txt configurado
- [ ] Tags canônicas corretas
- [ ] HTTPS ativado
- [ ] Amigável para dispositivos móveis
- [ ] Core Web Vitals aprovado
- [ ] Schema markup válido

## Checklist de Conteúdo SEO

- [ ] Meta títulos otimizados (50-60 caracteres)
- [ ] Meta descrições (150-160 caracteres)
- [ ] Hierarquia H1-H6 correta
- [ ] Estrutura de linkagem interna
- [ ] Textos alternativos de imagem

## Checklist GEO

- [ ] Seções de FAQ presentes
- [ ] Credenciais do autor visíveis
- [ ] Estatísticas com fontes
- [ ] Definições claras
- [ ] Citações de especialistas atribuídas
- [ ] Carimbos de data de "Última atualização"
